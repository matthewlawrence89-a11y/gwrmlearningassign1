# COMPAS Machine Learning Workflow (R → Python)

## Purpose
This project replicates an R-based machine learning workflow using Python. The analysis focuses on predicting recidivism risk and evaluating fairness across demographic groups.

## Methods
- Data preprocessing and filtering
- Exploratory Data Analysis (EDA)
- Logistic Regression modeling
- Model evaluation (accuracy, precision, recall)
- Fairness analysis (FPR and FNR by race)

## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- statsmodels
- scikit-learn

## How to Reproduce
1. Clone the repository
2. Install dependencies:
   pip install pandas numpy matplotlib seaborn statsmodels scikit-learn
3. Run the Jupyter Notebook

## Notes
This implementation mirrors the logic of the original R workflow while using Python libraries.
